# SleepyCat

SleepyCat — это лендинг-презентация приложения для сна и расслабления.  
Проект выполнен в пиксельном стиле и представляет функциональность приложения.
Сайт создан как демонстрация визуального стиля, верстки и работы с интерактивными элементами на чистом HTML, CSS и JavaScript.

---

## Возможности

- Пиксельный дизайн в стиле retro / pixel art  
- Адаптивная верстка
- Интерактивные элементы на JavaScript
- Страницы с описанием функций приложения (медитации, мантры, режим сна)
- Работа с изображениями и SVG

---

## Используемые технологии

- HTML5  
- CSS3 (основной упор на стили и визуал)
- JavaScript (интерактив и логика)
